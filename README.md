# Golden Spur Motor Inn Redesign

Production-ready Next.js rebuild for a premium upgrade redesign of the Golden Spur Motor Inn website.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run start
```

## Notes
- This project is structured for Vercel deployment.
- Content is based on publicly indexed business information from the original Golden Spur Motor Inn website and related listing snippets.
- Update metadata, favicon, and final photography/branding before launch.
